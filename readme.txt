This plugin will help you to display a sticky message in your site
   

== Description ==

This plugin will help the admin to publish highlighted notice on WordPress site header. Any message like offline notice, update notice, tweets, special offers, latest post can be set to attract visitors in 1st sight. 

= Features =
* Included tweets buttom (new)
* facebook like button (new)
* facebook fan page & tweeter profile url setup (new)
* Custom embed code support - you can use this option to place some link ads like google adsense (new)
* Custom Webfont support
* Custom background css option
* Fully customizable apearance and font
* Show/Hide option
* Very simple administration
 
== Installation ==


1. Upload `quick-notice` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
